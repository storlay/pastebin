## Описание

[pastebin.site](https://pastebin.site) — веб-приложение позволяющее загружать отрывки текста и делиться им с другими
пользователями

- docker — `bash` команды для запуска проекта
- drive — работа с API Google Drive, удаление и загрузка сообщений из/на Google Drive
- nginx — конфигурация `nginx`
- pastebin — конфигурация проекта
- text — создание / удаление текста сообщения
- users — работа с личным кабинетом пользователей
